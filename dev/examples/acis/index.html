ACIS Web Services¶

Examples of using Siphon’s simplewebservice support to access and use data from the Applied Climate Information System (ACIS) Web Services API provided by the Regional Climate Centers.

The ACIS Web Service API provides daily records for stations from the following networks: * WBAN * COOP * FAA * WMO * ICAO * GHCN * NWSLI/SHEF * ThreadEx * CoCoRahs * California Multi-station Index

Data is updated hourly and documentation on the API is available at http://www.rcc-acis.org/docs_webservices.html

A useful web tool for constructing query parameters is available here: http://builder.rcc-acis.org/
